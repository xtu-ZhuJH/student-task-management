# 项目计划书

## 一、项目名称
学生任务管理信息系统

## 二、项目背景
在日常学习过程中，学生常常需要处理课程作业、个人计划、考试安排等多项任务。传统记录方式存在信息分散、提醒不及时、管理效率低等问题。因此，开发一个轻量级任务管理信息系统，有助于提升个人任务规划与执行效率。

## 三、项目目标
开发一个完整的信息系统，实现用户管理、任务管理、公告管理和任务统计等功能，具备基本的数据增删改查能力，并形成完整的开发文档、测试文档和软件说明书。

## 四、功能模块
1. 用户注册与登录
2. 任务新增、修改、删除、查询
3. 公告发布与查看
4. 任务状态统计
5. 个人信息维护

## 五、技术方案
- 后端：Spring Boot
- 持久层：MyBatis
- 数据库：MySQL
- 前端：Thymeleaf + Bootstrap
- 版本管理：GitHub
