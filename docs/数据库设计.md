\# 学生任务管理系统 数据库设计（初版）\## 一、数据库概述数据库名称：student\_task\_db数据库版本：MySQL 8.0字符集：utf8mb4存储引擎：InnoDB\## 二、表结构设计\### 1. 系统用户表（sys\_user）存储管理员/学生登录账号信息| 字段名 | 类型 | 主键 | 非空 | 说明 || ---- | ---- | ---- | ---- | ---- || id | bigint | 是 | 是 | 用户ID || username | varchar(50) | 否 | 是 | 登录账号 || password | varchar(100) | 否 | 是 | 登录密码 || role | varchar(10) | 否 | 是 | 角色：admin/student || create\_time | datetime | 否 | 否 | 创建时间 |\### 2. 学生信息表（student）存储学生个人资料| 字段名 | 类型 | 主键 | 非空 | 说明 || ---- | ---- | ---- | ---- | ---- || id | bigint | 是 | 是 | 学生ID || user\_id | bigint | 否 | 是 | 关联用户ID || student\_no | varchar(20) | 否 | 是 | 学号 || name | varchar(20) | 否 | 是 | 姓名 || major | varchar(50) | 否 | 否 | 专业 || phone | varchar(11) | 否 | 否 | 手机号 |\### 3. 任务信息表（task）存储教学任务/作业信息| 字段名 | 类型 | 主键 | 非空 | 说明 || ---- | ---- | ---- | ---- | ---- || id | bigint | 是 | 是 | 任务ID || title | varchar(100) | 否 | 是 | 任务标题 || content | text | 否 | 否 | 任务内容 || publish\_time | datetime | 否 | 否 | 发布时间 || end\_time | datetime | 否 | 否 | 截止时间 || status | int | 否 | 否 | 状态：1启用 0禁用 |\## 三、设计说明1\. 采用三表分离设计，用户、学生、任务相互独立，关联清晰2\. 适配MySQL8.0，支持长期稳定运行3\. 结构简单，便于开发和后期扩展
